# Imágenes del proyecto

- El logo oficial debe colocarse en: `public/images/logo.png`
- Ruta esperada en Blade: `asset('images/logo.png')`
- Navbar, footer y home pública deben usar ese logo.
- Si el archivo no existe, la interfaz debe mostrar fallback visual sin romper la página.

## PWA
Los iconos específicos de la PWA viven en `public/images/pwa`.
